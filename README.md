NBA Salary Predictor (2000–2010)

This project builds a machine learning model that predicts NBA player salaries using player performance statistics from the 2000–2010 seasons.

Using historical data from Basketball-Reference, the project analyzes how on-court performance metrics influence player compensation in the National Basketball Association.

The pipeline includes:

Data collection and preprocessing of player statistics
Feature engineering using traditional and advanced basketball metrics
Training regression models to predict player salaries
Evaluating model performance and identifying key drivers of salary

The goal is to explore the relationship between player performance and financial valuation in professional basketball.

Features
Historical NBA player statistics (2000–2010)
Salary dataset merged with performance metrics
Machine learning models including:
Linear Regression
Random Forest
Gradient Boosting
Feature importance analysis
Salary prediction evaluation using RMSE and R²s
